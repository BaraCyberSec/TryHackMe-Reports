# TryHackMe Reports — BaraCyberSec

This repository documents my daily journey in **cybersecurity** and **SOC analysis** using TryHackMe.  
My goal is to build strong foundations (Networking, Linux/Windows, OSI, TCP/IP), practice **Blue Team** skills (log analysis, SIEM, incident response), and create a visible portfolio for future employers.

> One room a day. Learn by doing. Share the progress.

---

## Goals
- Become a **SOC Analyst L1** (short-term)
- Build habits: daily labs + short written reports
- Develop English technical writing skills
- Later transition into **ethical hacking**

---

## What’s inside
- **Weekly folders** with Markdown reports (EN, concise)
- Practical notes & mini-projects (personal security setup)
- Simple, repeatable templates to keep everything consistent

---

## How to use this repo
1. Open a week folder and read the room report(s).
2. Each report contains:
   - **Summary** (what the room teaches)
   - **Key Concepts**
   - **Tools/Commands**
   - **SOC Relevance**
   - **Personal Reflection**
3. Follow along or fork this repo to track your own learning.
